# CS-Mastery-Vault — Quick Start Guide

Welcome. This vault teaches you **how to learn computer science**, not computer science itself. It is built on peer-reviewed cognitive science, educational psychology, and expertise research.

## Installation

1. Unzip this archive anywhere on your computer.
2. Open **Obsidian** → "Open folder as vault" → select the unzipped `CS-Mastery-Vault` folder.
3. In Obsidian Settings → Community plugins:
   - Click "Browse" and install **Dataview** (then enable it).
   - Click "Browse" and install **Spaced Repetition** (then enable it).
   - The vault's pre-shipped `data.json` configurations for both plugins will be used automatically.
4. Open **[[00_MOCs/00 Master Index|Master Index]]** — your entry point.

## First 30 minutes

1. Read [[00_MOCs/How to Use This Vault|How to Use This Vault]].
2. Read [[01_Theory/T1 — Schema Transfer|T1 — Schema Transfer]].
3. Skim the [[00_MOCs/MOC — Schemas|10 schema dossiers]] — don't read them in depth; just see the landscape.
4. Open [[05_Roadmap/R1 — The 12-Month Study Sequence|R1 — the 12-month plan]].
5. Bookmark [[00_MOCs/Daily Dashboard|Daily Dashboard]] — this is your "today" view.

## The single most important habit

> End every study session by writing one retrieval prompt you cannot yet answer, and tag it `#sr`.

That one habit, sustained for six months, will outperform any other study strategy you adopt.

## The four vault rules

1. **Every note connects to a schema.**
2. **Every concept ends in retrieval.**
3. **Every session is logged.**
4. **Every week, abstract.**

## What this vault will and won't do

**Will**: compress years of inefficient self-study into a coherent system; eliminate redundant coverage; embed schema-based thinking; force retrieval and triage.

**Won't**: teach you the content of CS (that's your job); replace mentorship or real projects; make you a senior engineer in 12 months (no system can).
