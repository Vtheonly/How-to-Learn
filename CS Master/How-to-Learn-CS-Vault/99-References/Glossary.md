# 📖 Glossary

> *Key terms used throughout the vault.*

---

## A

**Abstraction ladder**
The range of abstraction levels at which a concept can be considered, from concrete instance to mathematical structure. See [[The-Abstraction-Ladder]].

**ACT-R**
Anderson's cognitive architecture modeling declarative and procedural memory. See [[Schema-Theory-and-Anderson]].

**Allostatic load**
The cumulative physiological cost of chronic stress. Elevated in burnout. See [[Burnout-Prevention]].

**Asymptotic notation**
O, Ω, Θ — used to describe algorithm complexity. See [[Learn-Algorithms-and-Data-Structures]].

---

## B

**Beacon**
High-information anchor in a codebase (entry point, type definition, public API) that reveals structure. See [[Reading-Codebases-Systematically]].

**Blocking practice**
Practicing one skill type at a time in long blocks. Less effective than interleaving. See [[Interleaving-and-Spacing-in-Practice]].

**Burnout**
Chronic physiological state from sustained over-reaching; characterized by anhedonia, sleep disturbance, cognitive impairment. See [[Burnout-Prevention]].

---

## C

**Cognitive load**
Mental effort imposed by a task; sum of intrinsic, extraneous, and germane load. See [[Cognitive-Load-Theory]].

**Cognitive Load Theory (CLT)**
Sweller's framework distinguishing intrinsic, extraneous, and germane load. See [[Cognitive-Load-Theory]].

**Concept map**
Visual network of concepts with labeled edges. See [[Concept-Mapping-Protocol]].

**Constraint-based analysis**
Triage heuristic predicting value by whether a work acknowledges domain constraints. See [[Constraint-Based-Analysis]].

**Criss-crossing**
Spiro's term for revisiting the same complex topic through multiple cases. See [[Cognitive-Flexibility-Theory]].

---

## D

**Deliberate practice**
Practice at the edge of competence with specific targets and feedback. See [[Deliberate-Practice]].

**Desirable difficulty**
A feature that makes practice harder but improves retention (e.g., spacing, interleaving, retrieval). See [[Interleaving-and-Spacing-in-Practice]].

**Disanalogy**
Where an analogy or isomorphism breaks down. As important as the analogy itself. See [[Isomorphism-Detection]].

---

## E

**Elaboration**
Cognitive process of adding detail, context, and connection to new information. See [[Elaboration-Strategies]].

**Extraneous load**
Cognitive load imposed by *how* material is presented, not what it is. Reducible. See [[Intrinsic-vs-Extraneous-Load]].

---

## F

**FLP impossibility**
Fischer-Lynch-Paterson theorem: deterministic consensus is impossible in async systems with one fault. See [[Learn-Distributed-Systems]].

**Forgetting curve**
Ebbinghaus's logarithmic decay of memory over time. See [[Spaced-Repetition-and-Forgetting]].

---

## G

**Germane load**
Cognitive load devoted to schema construction. Productive load to be maximized. See [[Intrinsic-vs-Extraneous-Load]].

---

## I

**Interleaving**
Mixing practice of different problem types within a session. Produces better retention than blocking. See [[Interleaving-and-Spacing-in-Practice]].

**Intrinsic load**
Cognitive load inherent to the material's difficulty. Manageable but not directly reducible. See [[Intrinsic-vs-Extraneous-Load]].

**Isomorphism**
Structure-preserving mapping between two domains. Enables transfer. See [[Isomorphism-Detection]].

---

## L

**Long-Term Working Memory (LT-WM)**
Ericsson & Kintsch's concept of using LTM as a working buffer via retrieval cues. See [[Long-Term-Working-Memory]].

---

## M

**MOC (Map of Content)**
A hub note that orients a section of the vault. See [[Home]] for the master MOC.

---

## R

**Retrieval practice**
Actively recalling information from memory; produces better retention than re-reading. See [[Testing-Effect-Retrieval-Practice]].

---

## S

**Schema**
Structured pattern in long-term memory capturing the typical features of a class of situations. See [[Schema-Theory-and-Anderson]].

**Schema-driven querying**
Reading with a stack of open questions rather than linearly. See [[Schema-Driven-Querying]].

**Selective ignorance**
The discipline of reserving deep cognitive investment for material that will compound. See [[Selective-Ignorance]].

**Spaced repetition**
Scheduling reviews at increasing intervals to optimize retention. See [[Spaced-Repetition-and-Forgetting]].

**Structure Mapping Theory (SMT)**
Gentner's framework for analogical transfer via structural alignment. See [[Structure-Mapping-Theory]].

**Syntopical reading**
Adler's method of reading multiple works on the same topic to construct a synthesis. See [[Syntopical-Reading]].

---

## T

**Testing effect**
Retrieval practice produces better retention than re-reading. See [[Testing-Effect-Retrieval-Practice]].

**Threshold concept**
A concept that, once understood, transforms the learner's view of the subject permanently. See [[Threshold-Concepts]].

**Triage**
The act of deciding what to read deeply, skim, or ignore. See [[Triage-Decision-Tree]].

---

## W

**Working memory (WM)**
Cognitive system for temporarily holding and processing information; ~4 chunks. See [[Cognitive-Load-Theory]].

**Working memory saturation**
State where WM is fully loaded and no further learning occurs; the "wall." See [[Working-Memory-Saturation]].
