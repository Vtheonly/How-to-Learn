# Master Index

> The single entry point to the entire vault. Bookmark this note.

This vault teaches you **how to learn computer science** using evidence-based cognitive science. Everything here is a mechanism for building two things:

1. **Schema density** — the number of reusable abstractions in your long-term memory.
2. **Retrieval strength** — how easily you can pull each schema back out under novel conditions.

| # | Principle | One-liner |
|---|-----------|-----------|
| T1 | [[01_Theory/T1 — Schema Transfer\|Schema Transfer]] | The brain moves knowledge between isomorphic domains via relational structure. |

| T3 | [[01_Theory/T3 — Deliberate Practice\|Deliberate Practice]] | Expertise is built by targeted, feedback-rich practice at the edge of ability. |
